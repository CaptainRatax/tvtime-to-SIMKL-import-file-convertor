How to find a SIMKL ID

Open SIMKL, search for the TV show, movie, or anime, and open its page. The SIMKL ID is the number in the URL after /tv/, /anime/, /movie/, or /movies/. Paste only that number in the SIMKL ID field.

SIMKL search is not always great. You can also search IMDb, copy the IMDb ID that starts with tt, and paste it into SIMKL search. SIMKL should open the matching page. Then copy the numeric SIMKL ID from that SIMKL page URL. Do not paste the IMDb ID into the SIMKL ID field.
